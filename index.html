ðŸ”— Blockchain Layer (EigenLayer AVSs & Ethereum L2)

Canvas Protocol is a Verifiable Application built with AVS technology on EigenLayer, operating its marketplace on an Ethereum L2. This provides:

  • Decentralized Trust & Security via EigenLayer Restaking:
    • AVS operators stake ETH, providing high economic security for validating interactions.
    • Attributable fault with slashing for misbehavior ensures operator honesty and the trustworthiness of attestations.
    • Shared security from Ethereum reduces risks compared to isolated systems.
  • Core Marketplace Operations on Ethereum L2:
    • Transparent B2B payments, ad auctions, and order matching.
  • Integrated AVS Services for Verifiability:
    • Decentralized Identity/Personhood AVS: Provides "Proof of Human Interaction" by validating CAPTCHA telemetry and user uniqueness (potentially via anonymous proof tokens or behavioral analysis), crucial for Phase 1 and beyond.
    • Data Validation AVS: For data collection CAPTCHAs (AI training, surveys), operators verify data quality/process compliance, generating on-chain attestations for clients.
    • Oracle AVSs: Securely bring verified interaction counts, campaign metrics, and settlement data on-chain for transparent marketplace operations and advertiser reporting.
    • (Future) Incentive Mechanism AVS: To manage and distribute Phase 2 user rewards based on verified interactions.
  • On-Chain Accountability: Irrevocable, transparent audit trail of interactions and attestations, building trust with all platform participants.

ðŸš€ Future Vision: Phase 2 - User Rewards & Ecosystem Growth

Once the B2B marketplace is established (Phase 1), Canvas Protocol will introduce direct crypto rewards (e.g., ETH/Stablecoins) to end-users for successfully completing interactive CAPTCHAs. This will create a three-sided economy, further incentivizing user participation, increasing the value of the CAPTCHA inventory for publishers, and providing an even more engaged audience for requesters. This phase aims to drive broader adoption of the Ethereum L2 and EigenLayer ecosystem and establish a new standard for user verification online.
